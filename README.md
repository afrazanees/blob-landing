# Blob Landing Page

An interactive animated landing page with GSAP animations, blob character, scramble text effects, and smooth scroll transitions.

## Features

- Animated blob character with interactive elements
- Scramble text effects and scribble SVG decorations
- Smooth scroll animations powered by GSAP
- Responsive design with Tailwind CSS
- Built with React, TypeScript, and Vite

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Tech Stack

- React 19.2
- TypeScript
- Vite
- GSAP with ScrollTrigger
- Tailwind CSS
- Radix UI components
